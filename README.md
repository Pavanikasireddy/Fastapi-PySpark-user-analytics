# FastAPI + PySpark User Analytics API

A RESTful API built with FastAPI and PySpark that performs analytics on user data.

### **Tech Stack**
- **Backend**: FastAPI, Uvicorn
- **Big Data**: Apache PySpark 
- **Language**: Python 3.10+

### **How to Run Locally**
```bash
git clone https://github.com/Pavanikasireddy/fastapi-pyspark-user-analytics.git
cd fastapi-pyspark-user-analytics
pip install -r requirements.txt
uvicorn main:app --reload
